# Lesson Plan Generator API

The Lesson Plan Generator is a FastAPI-based web application that uses AI to generate structured lesson plans for teachers. Educators can input simple prompts (e.g., “lesson plan for fifth graders on matter”) and receive detailed, curriculum-aligned lesson plans in seconds.

## 🚀 Features

AI-powered lesson plan generation
Fast and lightweight FastAPI backend
CORS-enabled for frontend integration
Deployed on Render for reliable hosting
Uses Hugging Face models for natural language generation

## 🧠 Tech Stack

 **Backend:** FastAPI
 **AI Model:** Hugging Face Hub
 **Deployment:** Render
 **Language:** Python 3.10+
